# Iterable iOS SDK

`iterable-ios-sdk` is an Objective-C implementation of an iOS client for Iterable.

# Automatic Installation (via CocoaPods)

Iterable supports [CocoaPods](https://cocoapods.org) for easy installation.

To install, add the pod to your podfile:
```
pod 'IterableSDK'
```

# Integration and Usage
